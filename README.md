# <img src="https://avatars.githubusercontent.com/u/175642905" width="32"> Somi API specification
![Swagger](https://img.shields.io/badge/Swagger-OAS_3.0-a?style=for-the-badge&logo=swagger&logoColor=black&labelColor=%2385ea2d&color=%23555555)

This repository holds the history of the Somi API specification. Each commit
is a version of the API specification, and the message contains both the version
number and the changelog. This means you can use the Git log to find when
certain endpoints were introduced or when features changed.
